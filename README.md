# 🏨 Rates & Inventory Management Platform

A comprehensive AI-powered Revenue Management platform designed for hotels, resorts, and hospitality businesses. Built with Next.js, TypeScript, and advanced analytics capabilities.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/shreykedia786/rates-inventory-management.git
cd rates-inventory-management
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Open application**
```
http://localhost:3000
```

## 🎮 Usage Guide

### Accessing Agentic AI
1. Click the **AI Insights** button in the header
2. Navigate between **Deep Insights**, **Analytics**, and **Automation** tabs
3. Review specific recommendations with confidence scores
4. Implement suggested actions with one-click workflows

### Managing Bulk Restrictions
1. Select multiple dates on the calendar
2. Open **Bulk Restrictions** popup
3. Configure rate and inventory settings
4. Use **Save Draft** or **Save & Publish** workflows

### Viewing Analytics
1. Access the **Analytics** tab in AI popup
2. Review revenue performance vs forecast
3. Analyze room-type and segment breakdowns
4. Monitor competitive positioning
